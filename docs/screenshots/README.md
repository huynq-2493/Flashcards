# Screenshots

Place your screenshots here with the following filenames:

| File | Content |
|------|---------|
| `dashboard.png` | Dashboard — due count, streak, retention |
| `study.png` | Study session — card front, show answer, rating buttons |
| `stats.png` | Statistics page — heatmap and charts |
| `decks.png` | Deck list page |
| `cards.png` | Card management inside a deck |
| `import.png` | CSV bulk import modal |

You can take screenshots by running the app locally or with Docker and capturing each page.
